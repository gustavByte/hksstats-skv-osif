# Security Policy

Hvis du oppdager et sikkerhetsproblem i dette repoet eller på nettsiden, ikke legg det i et offentlig issue først.

Send heller en privat melding til repo-eier via GitHub:

- https://github.com/gustavByte

Beskriv gjerne:

- hva som er problemet
- hvordan det kan reproduseres
- hvilken URL, fil eller funksjon som er berørt
- om problemet allerede er offentlig kjent

Målet er å bekrefte mottak raskt og få en rettelse ut før detaljer deles offentlig.
